# 📸 Image Feed App

A full-stack MERN application I built to practice real-world image handling using Multer and MongoDB



## 🚀 Features
- Upload images with title/captions
- View all images in a feed
- Delete images
- Responsive design

## 🛠️ Tech Stack
| Frontend | Backend | Database |
|----------|---------|----------|
| React.js | Node.js | MongoDB  |
| CSS      | Express.js | Mongoose |
|          | Multer  | |
